# 知己 - 产品文档

> "越聊越懂你的人生伙伴"

---

## 一、产品愿景

打造一个**真正懂你**的 AI 人生伙伴 + 职业规划顾问。

不是算命工具，不是角色扮演，而是：

- 基于命理建立深度画像
- 通过日常互动持续理解你
- 在人生困惑时给出有分量的参考
- **提供个性化的职业规划指导**

**核心解决的问题**：现代人的焦虑与迷茫（尤其是职业焦虑），需要一个不评判、懂你、能给启发的存在。

## 四、核心功能模块

### 4.1 命盘画像（首次使用）

**输入**：出生日期、时辰、出生地

**输出**：

- 性格底色分析
- 思维模式特点
- 易踩的坑/盲点
- 天赋优势方向
- **职业天赋图谱**（适合的职业类型与方向）

**设计原则**：

- 不说"命中注定"，说"你这种特质的人往往..."
- 可视化呈现，美观易懂
- 可随时回顾

---

### 4.2 智者对话（核心功能）

**定位**：随时可聊的人生伙伴

**智者人设**：

- 身份：阅历丰富的智者
- 风格：温和、不急躁、偶尔幽默
- 原则：不给标准答案，给视角；不评判，只陪伴
- 知识体系：东方智慧 + 西方哲学 + 心理学

**对话特色**：

- 慢回复模式（可选）：3-5 秒延迟，增加分量感
- 深度记忆：记住你之前说过的所有事
- 命理融入：在关键时刻点出你的特质

**示例对话**：

```
用户：最近很焦虑，不知道要不要跳槽

智者：你急着要答案，但有时候问题本身比答案重要。
     你说怕选错——你怕的是错，还是怕后悔？

     你命里印星重，天生容易困在"万一"里。
     但你这种人一旦真正决定了，执行力反而很强。

     也许问题不是"跳不跳"，而是"你在怕什么"？
```

---

### 4.3 状态追踪

**自动记录**：

- 每次对话的情绪基调
- 困惑主题分类
- 状态变化趋势

**系统洞察**：

- "你最近三周都在焦虑职业方向"
- "每到月底你的焦虑会加重"
- "比起上个月，你提到 XX 的次数少了"

---

### 4.4 周度/月度回顾

**每周自动生成**：

- 本周情绪曲线
- 主要困惑主题
- 成长与变化
- 一句话总结

**月度报告**：

- 长期趋势分析
- 命盘与实际状态的呼应
- 下月时运参考

---

### 4.5 智慧卡片

**每次有价值的对话后**：

- 提炼一句话启发
- 形成可收藏的卡片
- 积累成你的专属"智慧语录库"

**示例**：

> "你以为在选工作，其实在选一种生活方式。"
> — 2026.01.13 关于跳槽的对话

---

### 4.6 决策参考（关键时刻）

**当用户面临重大决策时**：

输入具体问题 + 背景信息

输出：

- 利弊分析
- 基于你性格的盲点提醒
- 时机判断（结合流年）
- 不同选择的可能走向
- 一句话建议

**特点**：基于长期对话积累，不是泛泛而谈

---

### 4.7 时机提醒

**结合命盘流年**：

- 推送适合行动的时机
- 提示需要注意的阶段
- 节气/时令融入，增加仪式感
- **职业窗口期提醒**（跳槽/升职/转型最佳时机）

**示例**：

> "本月中旬适合重要谈话，你一直犹豫的那件事，可以考虑行动了。"
> "今年下半年是你的事业上升期，可以主动争取更多机会。"

---

### 4.8 职业规划（特色模块）

**定位**：不是泛泛的职业测试，而是**基于命理画像的个性化职业指导**

#### 4.8.1 职业天赋图谱

**基于命盘分析**：

- 你适合的职业**类型**（而非具体岗位）
- 例：适合"需要深度思考的工作"而非"程序员"
- 你的职业优势与潜在短板
- 最能发挥你的工作环境类型

**呈现方式**：

- 可视化雷达图/图谱
- 不是"命中注定"，而是"你这种特质适合..."

#### 4.8.2 职业匹配度分析

**用户输入**：目标职业/岗位/行业

**系统输出**：

- 匹配度评分（0-100）
- 你的优势在这个岗位如何发挥
- 你需要注意的潜在挑战
- 是否适合长期发展
- 具体建议

**示例**：

```
目标：产品经理

匹配度：78分

优势：
- 你善于系统思考，适合把控产品全局
- 你的沟通特质适合跨部门协调

注意：
- 你命里印星重，容易陷入细节，要学会抓大放小
- 你不太擅长"快速决策"，产品经理需要在这方面刻意练习

建议：
- 适合偏策略型的产品方向，而非快节奏的增长型产品
```

#### 4.8.3 职业时间线

**可视化展示**：

- 你的职业关键窗口期（未来 1-5 年）
- 适合跳槽/转型的时机
- 适合蛰伏积累的阶段
- 可能面临的挑战期

**交互方式**：

- 时间轴可点击查看详情
- 每个阶段有具体建议

#### 4.8.4 职业决策助手

**场景化问答**：

- "这个 offer 要不要接？"
- "要不要跳槽？"
- "要不要转行？"
- "什么时候提升职/加薪？"

**输出**：

- 基于你的命盘特质分析
- 结合当前时运判断
- 利弊对比
- 时机建议
- 具体行动指南

## 七、技术架构

### 7.1 技术栈

| 模块       | 技术选型                   |
| ---------- | -------------------------- |
| iOS 客户端 | SwiftUI                    |
| 后端服务   | Python (FastAPI) / Node.js |
| AI 对话    | Claude API / GPT-4 API     |
| 数据库     | PostgreSQL + Redis         |
| 云服务     | AWS / 阿里云               |

### 7.2 核心技术要点

**命理计算引擎**：

- 八字排盘算法
- 流年大运计算
- 性格特质映射规则

**AI 对话系统**：

- 智者人设 Prompt 设计
- 长期记忆管理（向量数据库）
- 对话上下文压缩与召回

**个性化引擎**：

- 用户画像持续更新
- 对话主题分类
- 情绪识别与追踪

## 八、UI/UX 设计方向

### 设计风格：**简约东方美学**

- 配色：淡雅自然（米白、墨绿、淡金点缀）
- 布局：圆润卡片式，呼吸感强
- 字体：优雅有质感
- 动画：流畅克制，有仪式感
- 支持深色/浅色主题

### 核心界面

1. **首页**：智者对话入口 + 今日一句
2. **命盘页**：可视化个人画像 + 职业天赋图谱
3. **职业页**：职业时间线 + 岗位匹配测试 + 决策助手
4. **档案页**：成长轨迹 + 智慧卡片库 + 职业决策记录
5. **回顾页**：周度/月度报告

## 十一、风险与应对

| 风险                 | 应对                             |
| -------------------- | -------------------------------- |
| 用户把它当算命工具   | 产品文案强调"参考"而非"预测"     |
| AI 回复质量不稳定    | 精心设计 Prompt + 持续优化       |
| 命理争议（封建迷信） | 定位为"性格分析工具"，加免责声明 |
| 用户隐私担忧         | 本地存储敏感信息 + 隐私政策透明  |
